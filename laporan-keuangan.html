Kebijakan Privasi

Aplikasi: Personal: Laporan Keuangan

Terakhir Diperbarui: 16 Maret 2026

Selamat datang di Personal: Laporan Keuangan. Kami sangat menghargai privasi Anda dan berkomitmen untuk melindungi informasi keuangan Anda secara aman dan transparan.

Penting: Aplikasi ini dirancang dengan prinsip privasi utama. Kami tidak mengumpulkan atau menyimpan data transaksi keuangan Anda di server kami. Seluruh data transaksi adalah hak dan milik penuh pengguna.

1. Informasi yang Kami Kumpulkan

Dalam menjalankan fungsinya, aplikasi dapat mengakses atau memproses jenis informasi berikut:

  • Data Transaksi (Lokal): Catatan pemasukan, pengeluaran, piutang, dan hutang yang Anda input. Data ini disimpan hanya di dalam memori perangkat Anda.
  • Informasi Identitas: Alamat email digunakan untuk keperluan login Google dan sinkronisasi cadangan.
  • Informasi Teknis: ID Perangkat (Device ID), ID Iklan (Advertising ID), dan log performa (Crash logs) untuk memastikan aplikasi berjalan stabil.

2. Penggunaan dan Penyimpanan Data

  • Penyimpanan Lokal: Seluruh data keuangan Anda bersifat privat dan tetap berada di perangkat Anda. Kami tidak memiliki akses untuk melihat atau membaca data transaksi tersebut.
  • Fitur Cadangan (Backup): Jika Anda mengaktifkan fitur Backup, data akan dikirim ke akun Google Drive pribadi Anda menggunakan enkripsi HTTPS. Data tersebut disimpan di ruang penyimpanan cloud milik Anda sendiri, bukan di server kami.
  • Layanan Iklan: Kami menggunakan Google AdMob untuk menampilkan iklan guna mendukung pengembangan aplikasi.

3. Keamanan Informasi

Kami menerapkan standar keamanan tinggi:

  • Enkripsi saat Transit: Semua transmisi data (seperti proses backup atau pelaporan error) dilakukan melalui koneksi terenkripsi HTTPS/TLS yang aman.
  • Kontrol Pengguna: Anda memiliki kendali penuh untuk menghapus seluruh data di aplikasi maupun file cadangan di Google Drive kapan saja.

4. Layanan Pihak Ketiga

Aplikasi ini berinteraksi dengan layanan resmi dari Google untuk menunjang fungsionalitas:

  • Google Play Services & Drive API: Untuk manajemen akun dan backup.
  • Google AdMob: Untuk penayangan iklan yang relevan.
  • Firebase Crashlytics: Untuk analisis kerusakan aplikasi secara anonim.
